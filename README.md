# choo-example

> An example of choo application

## Install

```sh
git clone https://github.com/andrepolischuk/choo-example
cd choo-example
npm install
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
